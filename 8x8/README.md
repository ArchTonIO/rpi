# 8x8

Adafruit LED 8x8 green matrix with I2C pack using HT16K33 controller

![](../8x8.jpg)

```
git clone https://github.com/adafruit/Adafruit_LED_Backpack
git clone https://github.com/adafruit/Adafruit-GFX-Library
```

Connection:
```
     8x8    Uno
+      +     5V
-      -    GND
SDA    D     A4
SCL    C     A5
```
