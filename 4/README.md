# 4 device

![](../4.jpg)

Install libraries:
```
git clone git@github.com:blippy/lcddogmspi.git
```

Low-level controllers:
* `lcdard` - Nano/Uno
* `lcdesp.py` - ESP8266
* `lcdpi.py` - RPi3. Use this is as a baseline
