# stm8s103f3p6

Inbuilt LED is B5

8kB Program Memory (Flash), 1kB RAM, and 640Byte EEPROM.

STM8S103F3P6 contains 1 x 8bit and 2 x 16bit timer, 16 I/Os, thereof 5 analog inputs 10bit, 1 x SPI, 1 x I2C, 1 x UART.
