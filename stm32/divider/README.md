# clock/frequency divider

PB12 is input pin. The output pins are as follows:

```
GPIO DIVISOR
PB13       2
PB14       4
PB15       8
PA8       16
PA9       32
PA10      64
PB5      128
PB6      256
PB7      512
PB8     1024
PB9     2048
```

The output pins are clocked on rising edges of the input pin.
